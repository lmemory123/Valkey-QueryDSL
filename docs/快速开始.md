# 快速开始

这篇文档的目标是让你在一个新的 Spring Boot 项目中，把 `Valkey QueryDSL` 接入并跑通第一条查询。

## 1. 前置条件

- Java 17+
- Maven 3.9+
- 可用的 Valkey 服务
- 服务端具备 Search / JSON 能力

## 2. 安装当前版本

如果你要在其他项目中引用当前仓库版本，先在本仓库根目录执行：

```bash
mvn clean install
```

当前版本会安装到本机 Maven 仓库：

- `com.momao:valkey-query-annotations:1.0.1-SNAPSHOT`
- `com.momao:valkey-query-core:1.0.1-SNAPSHOT`
- `com.momao:valkey-query-processor:1.0.1-SNAPSHOT`
- `com.momao:valkey-query-glide-adapter:1.0.1-SNAPSHOT`
- `com.momao:valkey-query-spring-boot-starter:1.0.1-SNAPSHOT`

## 3. 添加依赖

业务项目至少需要引入 Starter：

```xml
<dependency>
    <groupId>com.momao</groupId>
    <artifactId>valkey-query-spring-boot-starter</artifactId>
    <version>1.0.1-SNAPSHOT</version>
</dependency>
```

并为编译器配置 APT：

```xml
<plugin>
    <groupId>org.apache.maven.plugins</groupId>
    <artifactId>maven-compiler-plugin</artifactId>
    <configuration>
        <annotationProcessorPaths>
            <path>
                <groupId>com.momao</groupId>
                <artifactId>valkey-query-processor</artifactId>
                <version>1.0.1-SNAPSHOT</version>
            </path>
        </annotationProcessorPaths>
    </configuration>
</plugin>
```

## 4. 启用扫描

```java
@SpringBootApplication
@EnableValkeyQuery(basePackages = "com.example.demo")
public class DemoApplication {
}
```

## 5. 配置连接

```yaml
valkey:
  query:
    enabled: true
    use-tls: false
    request-timeout: 5000
    connection-timeout: 5000
    username: your-username
    password: your-password
    nodes:
      - host: 127.0.0.1
        port: 6379
```

## 6. 定义实体

```java
@ValkeyDocument(
    indexName = "idx:sku",
    prefixes = {"sku:"},
    storageType = StorageType.JSON
)
public class Sku {

    @ValkeyId
    private String id;

    @ValkeySearchable(weight = 2.5d, noStem = true)
    private String title;

    @ValkeyIndexed(sortable = true)
    private Integer price;

    @ValkeyIndexed
    private List<String> tags;

    @ValkeyIndexed
    private Merchant merchant;
}
```

```java
public class Merchant {
    private String name;
    private String level;
}
```

编译后会生成 `SkuQuery`。

## 7. 定义 Repository

```java
@Repository
public class SkuRepository extends BaseValkeyRepository<Sku> {

    public SkuRepository(GlideClient glideClient, ObjectMapper objectMapper) {
        super(SkuQuery.METADATA, glideClient, Sku.class, objectMapper);
    }
}
```

## 8. 发起查询

```java
SkuQuery q = new SkuQuery();

Page<Sku> page = skuRepository.queryChain()
    .where(q.title.contains("iphone"))
    .and(q.merchant.name.eq("Apple Flagship"))
    .orderByAsc("price")
    .page(0, 20);
```

或者直接使用 Repository：

```java
SearchCondition condition = q.title.contains("iphone").sortBy("price", true);

List<Sku> list = skuRepository.list(condition);
Page<Sku> result = skuRepository.page(condition, 0, 20);
Sku one = skuRepository.one(q.id.eq("sku-1"));
long total = skuRepository.count(q.tags.eq("HOT"));
```
